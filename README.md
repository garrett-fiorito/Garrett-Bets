# Bet Tracker

Dark React/Supabase bet tracker for singles, futures, and parlays.

## Setup

1. Create a Supabase project.
2. Run `supabase/schema.sql` in the Supabase SQL editor.
3. Copy `.env.example` to `.env.local` and add your Supabase URL and anon key.
4. Run:

```bash
npm install
npm run dev
```

Deploy on Vercel or Netlify with the same environment variables.
